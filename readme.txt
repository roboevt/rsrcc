C-- compiler for the RSRC CPU (Really Simple RISC (Reduced Instruction Set CPU (Central Processing Unit)))

Compiles C-- code to SRC assembly. (https://www.iro.umontreal.ca/~dift1213/Slides/Ch02.pdf)

Known limitations of C--:
Arguments must be variables.
